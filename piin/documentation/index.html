FOLDER STRUCTURE

After downloading and extracting, you will find 2 folders :

  • Documentation : contain all documentation files.
  • Website : contain all template files.

In website folder you will find these folders :

  • css : contain css files for themes and animations (css3 animation).
  • fonts : contain all font files.
  • img : contain all image and icon files.
  • js : contain javascript files.
  • php : contain all php script files.

The main html file are :

  • demo_1.html : demo with slideshow background.
  • demo_2.html : demo with kenburns slideshow background.
  • demo_3.html : demo with single image + firefly effect background.
  • demo_4.html : demo with YouTube video background.
  • demo_5.html : demo with single image + star effect background.
  • demo_6.html : demo with self hosted video background.

You can find the main html file in website folder.

JQUERY FULLPAGE

We are using jQuery fullpage in about-section and contact-section. They will active when the about-section or contact-section is focused and they will be inactive when the page focused in home-section. You can see how to activate and deactivate them in main.js file at function : SlideToAbout(), SlideToContact(), and SlideToHome().

Basically, this is how to activate the jQuery fullpage plugin 

$('.contact-section').fullpage({
    anchors: ['contact-details','send-message','newsletter'],
    sectionsColor: [],
    navigation: false,
    navigationPosition: 'right',
    navigationTooltips: [],
    slideSelector: '.fullpage-slide',
    scrollOverflow: true,
    sectionSelector: '.fullpage-section',
    easingcss3: 'cubic-bezier(0.175, 0.885, 0.320, 1.275)'
});

And this is how to deactivate them 

$.fn.fullpage.destroy('all');

In this template, we only activate the plugin on medium and large devices in order to keep the template stay look beautifully in small devices.

More documentation about this plugin, please visit https://github.com/alvarotrigo/fullPage.js

FIREFLY BACKGROUND EFFECT

We are using jQuery firefly plugin to enabling the firefly background effect. To activate it, put this code inside document.ready function in main.js file. 

if($(".bg-container-firefly").is(':visible')){
    $(".bg-container-firefly").backstretch([
        "img/slideshow/sample5.jpg"    //-- CHANGE WITH YOUR IMAGE URL
    ],{
        duration:6000,
        fade:'normal'
    });
			
    $.firefly({
        color: '#646464',	//-- firefly color
        minPixel: 1,					
        maxPixel: 3,
        total : 70,
        on: '.bg-container-firefly'
    });
}

Insert this code after the <body> tag. 

<div class="bg-container-firefly"></div>

The last, import firefly script. Put it before the </body> tag. 

<script type="text/javascript" src="js/jquery.firefly.min.js"></script>

More documentation about this plugin, please visit https://github.com/motyar/firefly

SLIDESHOW BACKGROUND

We are using Backstretch Plugin to enabling the slideshow background. To activate it, first insert this line after the <body> 

<div class="bg-container-slideshow"></div>

Then put this code inside document.ready function in main.js file. 

if($(".bg-container-slideshow").is(':visible')){
    $(".bg-container-slideshow").backstretch([
        "img/slideshow/sample1.jpg",	//-- CHANGE WITH YOUR IMAGE URL
        "img/slideshow/sample2.jpg"
    ],{
        duration:6000,
        fade:'normal'
    });
}

If you want to make it static, just insert one image url. Then import jQuery backstretch script, put it before </body> tag. 

<script type="text/javascript" src="js/jquery.backstretch.min.js"></script>

More documentation about this plugin, please visit http://srobbin.com/jquery-plugins/backstretch/

YOUTUBE VIDEO BACKGROUND

We are using jquery-mb-ytplayer plugin to enabling the YouTube video background. To activate it, first put this code after <body>. 

<a id="video" class="player" data-property="{videoURL:'2e52RcPAlwU',containment:'.bg-container-youtube', showControls:false, autoPlay:true, loop:true, mute:true, startAt:0, opacity:1, addRaster:false, quality:'large'}"></a>
        
<div class="bg-container-youtube"></div>

Replace the videoURL in data-property with yours, replace with the last part of your YouTube video URL. Then import jQuery YTPlayer script, put it before the </body> tag. 

<script type="text/javascript" src="js/jquery.mb.YTPlayer.js"></script>

The last, insert this code inside document.ready function in main.js file. 

if($(".bg-container-youtube").is(':visible')){
    if($(window).width() >= 1200){
        $('.player').mb_YTPlayer();
    }
    else{
        $(".bg-container-youtube").backstretch([
            "img/slideshow/sample.jpg",		//-- CHANGE WITH YOUR IMAGE URL
            "img/slideshow/sample2.jpg"
            ],{
            duration:6000,
            fade:'normal'
        });
    }
}

Please note that this player doesnâ€™t work as background for devices due to the restriction policy adopted by all on managing multimedia files via javascript. It fallsback to the default Youtube player if used as player (not applied to the body). If fallback we've just set it up to slideshow background using backstretch, just replace the image in backstretch initialization with yours.

More documentation about this plugin, please visit http://pupunzi.open-lab.com/mb-jquery-components/jquery-mb-ytplayer/

SELF HOSTED VIDEO BACKGROUND

We are using jQuery backgroundvideo to enabling the self hosted video background. To activate it, do these steps :

Insert this line after <body> tag. 

<div class="bg-container-video">
</div>

Import jquery.backgroundvideo.js, put it before the </body> tag. 

<script type="text/javascript" src="js/jquery.backgroundvideo.js"></script>

Then put this code inside document.ready function in main.js file. 

if($(".bg-container-video").is(':visible')){
    var videobackground = new $.backgroundVideo($('.bg-container-video'), {
        "align": "centerXY",
        "width": 1280,                      //-- proportion (doesnt need to change)
        "height": 720,                      //-- proportion (doesnt need to change)
        "path": "img/video/",               //-- video folder path
        "filename": "video_placeholder",    //-- video name
        "types": ["mp4"]                    //-- video file type
    });
}

KENBURNS SLIDESHOW BACKGROUND

We are using jQuery Kenburnsy Plugin to enabling the kenburns slideshow background. To activate it, do these steps :

<link rel="stylesheet" href="css/jquery.kenburnsy.css" type="text/css">

Import jquery.velocity.min.js and jquery.kenburnsy.js, put it before the </body> tag. 

<script type="text/javascript" src="js/jquery.velocity.min.js"></script>
<script type="text/javascript" src="js/jquery.kenburnsy.js"></script>

Insert this line after <body> tag. 

<div class="bg-container-kenburns">
    <img src="img/slideshow/sample3.jpg" alt="image1" />
    <img src="img/slideshow/sample4.jpg" alt="image2" />
</div>

Replace the images with yours, you can add more images if you want more than two images slideshow.

Then put this code inside document.ready function in main.js file. 

if($(".bg-container-kenburns").is(':visible')){
    $(".bg-container-kenburns").kenburnsy({
        fullscreen: true
    });
}

More documentation about this plugin, please visit https://github.com/ZeroOneStudio/kenburnsy

FORM VALIDATION

                            

                       

                       
                       
                       
                       

                       		

                            	

                                	We are using jQuery Validate Plugin to enabling form validation in subscribe form and contact form. To activate it, insert this code inside document.ready function in main.js file.
                                    
//-- validate and submit subscribe form
$('.subscribe-form').validate({
    errorLabelContainer: $('.subscribe-notif'),
    rules: {
        EMAIL: {
            required: true,
            email: true
        }
    },
    messages: {
        EMAIL: {
            required: "Please insert your email address",
            email: "Your email address is not valid"
        }
    },
    submitHandler: function(form) {							
        var url_dest = $(form).attr('action');
        var form_data = $(form).serialize();
        var form_method = $(form).attr('method');
			
        //-- show loading
        $('.subscribe-notif').show().html('<label class="loading-subscribe">Please wait</label>');
        $('.loading-subscribe').fadeIn('fast');
			
        $.ajax({
            type: form_method,
            url: url_dest,
            data: form_data,
            cache : false,
            dataType : 'json',
            contentType: "application/json; charset=utf-8",
            error : function(err) { alert("Could not connect to the registration server. Please try again later."); },
            success : function(data) {
                if(data.result == "success"){
                    //-- reset form
                    $(form).trigger('reset');
							
                    //-- set element to focusout
                    $('.subscribe-email').focusout();
                        
                    //-- hide loading
                    $('.loading-subscribe').fadeOut('fast',function(){
                        //-- show notif
                        $('.subscribe-notif').html('<label class="subscribe-notif-success">Thank you for subscribing us.</label>');
                        $('.subscribe-notif-success').fadeIn('fast').delay(5000).fadeOut('fast',function(){
                            $(this).remove();
                            $('.loading-subscribe').remove();
                        });
                    });
                }
                else{
                    //-- reset form
                    $(form).trigger('reset');
					
                    //-- hide loading
                    $('.loading-subscribe').fadeOut('fast',function(){
                        //-- show notif
                        $('.subscribe-notif').html('<label class="subscribe-notif-error">Error.</label>');
                        $('.subscribe-notif-error').fadeIn('fast').delay(5000).fadeOut('fast',function(){
                            $(this).remove();
                            $('.loading-subscribe').remove();
                        });
                    });
                }
            }
        });
							
        return false;
    }
});
//-- end validate and submit subscribe form
		
		
//-- validate and submit contact us form
$('.contact-form').validate({
    rules: {
        email: {
            required: true,
            email: true
        },
        name: {
            required: true
        },
        message: {
            required: true
        }
    },
    highlight: function(element, errorClass, validClass) {
        $(element).addClass('form-error');
    },
    unhighlight: function(element, errorClass, validClass) {
        $(element).removeClass('form-error');
    },
    errorPlacement: function(error, element) {
    },
    submitHandler: function(form) {
        var url_dest = $(form).attr('action');
        var form_data = $(form).serialize();
				
        //-- show loading
        $('.contact-notif').show().append('<label class="loading-contact">Please wait</label>');
        $('.loading-contact').fadeIn('fast');
			
        $.post(url_dest,form_data,function(data){
            var success = data;
					
            if(success){
                //-- reset form
                $(form).trigger('reset');
					
                //-- hide loading
                $('.loading-contact').fadeOut('fast',function(){
                    //-- show notif
                    $('.contact-notif').append('<label class="contact-notif-success">Thank you for contacting us. We will reply you shortly.</label>');
                    $('.contact-notif-success').fadeIn('fast').delay(5000).fadeOut('fast',function(){
                        $(this).remove();
                        $('.loading-contact').remove();
                    });
                });
            }
            else{
                //-- reset form
                $(form).trigger('reset');
						
                //-- hide loading
                $('.loading-contact').fadeOut('fast',function(){
                    //-- show notif
                    $('.contact-notif').append('<label class="contact-notif-error">Error.</label>');
                    $('.contact-notif-error').fadeIn('fast').delay(5000).fadeOut('fast',function(){
                        $(this).remove();
                        $('.loading-contact').remove();
                    });
                });
            }
        });
				
        return false;
    }
});
//-- end validate and submit contact us form
